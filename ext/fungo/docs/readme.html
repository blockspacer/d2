Instructions

Thanks for downloading fungo, a C++ library that enables you to store and later re-throw exceptions caught inside a catch(...) block. This is particularly useful when your C++ implementation does not support std::exception_ptr and you have a need to propagate exceptions out of asynchronous tasks, or carry them across a C/C++ boundary interface.

Please visit http://bitbucket.org/edd/fungo/wiki/Home for more information on fungo, including compilation and usage instructions.

— Edd
